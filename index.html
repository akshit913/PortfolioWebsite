Recent Projects

COVID-19 Speech Agent

Speech chatbot which provides the statistical information (number of cases) regarding the current COVID-19 pandemic. The user can ask the bot about the deaths and confirmed, recovered cases for any country, state or county.
The program is developed using Google's Dialogflow and Unity 3D.
Dialogflow provides natural speech recognition tools which can be used to extract the keywords and to understand the intent of the user. Google's fullfilment was also used to bridge the logic and the user input.
Unity 3D was used to develop the 3D front end and the character was chosen from mixamo.

Recent Projects

Hell's Chicken - 2D Android Platform Game

2D platform shooter game developed with the aim of incorporating natural , interactive gestures to make the gameplay more exciting for the user.
The game uses directional gestures to prevent the restrictiveness caused by the buttons.
The user can shoot the enemies with a set of different shots which can be fired by performing the assigned gestures.
Both shooting and movement gestures can be applied simultaenously.
The story of the game is about a rabbit which enters hell with an empty stomach and needs to fight the enemies to grab the chicken present at the end of the level before starving to death.
